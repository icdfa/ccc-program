# CCC Program Pre-Assessment

## Instructions

This pre-assessment is designed to help you gauge your existing knowledge across the core domains of the Certified Cybersecurity Consultant (CCC) program. It consists of 20 multiple-choice questions. Please answer to the best of your ability. The answer key is provided at the end.

---

### Questions

**1. What is the primary purpose of a Statement of Work (SOW) in a consulting engagement?**
   a) To serve as a marketing brochure
   b) To legally bind the client to a long-term contract
   c) To define the scope, deliverables, and terms of the engagement
   d) To provide a detailed technical implementation guide

**2. Which of the following best describes the principle of "least privilege"?**
   a) All users should have administrative rights
   b) Users should only have the minimum access necessary to perform their job functions
   c) Access should be granted based on seniority
   d) All access should be logged and monitored

**3. In the context of risk management, what is "residual risk"?**
   a) The risk before any controls are applied
   b) The risk that remains after controls are applied
   c) The risk that is transferred to a third party
   d) The risk that is accepted by the organization

**4. Which of the following is a key component of the NIST Cybersecurity Framework (CSF)?**
   a) Identify, Protect, Detect, Respond, Recover
   b) Plan, Do, Check, Act
   c) Confidentiality, Integrity, Availability
   d) Governance, Risk, Compliance

**5. What is the primary purpose of a Statement of Applicability (SoA) in ISO 27001?**
   a) To list all possible security controls
   b) To document which controls from Annex A are applicable and why
   c) To serve as a contract with the certification body
   d) To define the scope of the ISMS

**6. Under GDPR, what is the "right to be forgotten" also known as?**
   a) Right to access
   b) Right to rectification
   c) Right to erasure
   d) Right to data portability

**7. Which of the following is a primary goal of a security audit?**
   a) To implement new security controls
   b) To provide an independent assessment of control effectiveness
   c) To respond to security incidents
   d) To develop security policies

**8. What is the primary difference between a security architect and a security engineer?**
   a) Architects design, engineers build
   b) Engineers design, architects build
   c) There is no difference
   d) Architects focus on physical security, engineers on logical security

**9. Which cloud service model provides the highest level of control to the customer?**
   a) SaaS (Software as a Service)
   b) PaaS (Platform as a Service)
   c) IaaS (Infrastructure as a Service)
   d) FaaS (Function as a Service)

**10. What is the primary goal of a Zero Trust Architecture?**
   a) To trust all internal network traffic
   b) To eliminate the need for firewalls
   c) To never trust, always verify, and assume breach
   d) To rely solely on perimeter security

**11. What is the main purpose of a Business Impact Analysis (BIA)?**
   a) To identify the financial impact of a disaster
   b) To determine the recovery time objectives (RTOs) and recovery point objectives (RPOs) for critical business functions
   c) To test the disaster recovery plan
   d) To select a disaster recovery site

**12. In executive communication, what is the "Pyramid Principle"?**
   a) Starting with the details and building to a conclusion
   b) Starting with the answer/recommendation first, then providing supporting details
   c) Using complex charts and graphs
   d) Focusing on technical jargon

**13. What is "scope creep" in project management?**
   a) The process of defining the project scope
   b) The uncontrolled expansion of project scope without adjustments to time, cost, and resources
   c) The successful completion of the project scope
   d) The initial phase of project planning

**14. Which of the following is a key component of a capstone project?**
   a) A final exam
   b) A comprehensive, end-to-end simulated project
   c) A series of multiple-choice quizzes
   d) A group presentation

**15. Which of the following is NOT a domain of the CISSP certification?**
   a) Security and Risk Management
   b) Asset Security
   c) Software Development Security
   d) Financial Auditing

**16. What is the primary focus of the CISM certification?**
   a) Technical penetration testing
   b) Information security governance and management
   c) Secure software development
   d) Network forensics

**17. Which of the following is a key principle of the NICE Framework?**
   a) To provide a common language for cybersecurity work
   b) To mandate specific security controls
   c) To certify cybersecurity professionals
   d) To develop security software

**18. What is the primary purpose of a code of conduct in a professional community?**
   a) To list all possible legal violations
   b) To establish professional standards and create a respectful environment
   c) To define the technical requirements for participation
   d) To serve as a marketing tool

**19. In the context of a GitHub repository, what is a "pull request"?**
   a) A request to download the repository
   b) A notification of a security vulnerability
   c) A proposed change to the repository from a contributor
   d) A request for technical support

**20. What is the primary benefit of using a version control system like Git?**
   a) It automatically backs up all files to the cloud
   b) It tracks changes to files over time and allows for collaboration
   c) It encrypts all files by default
   d) It prevents unauthorized access to files

## Usage Guidelines

### Print Quality

All flyers are generated at high resolution suitable for professional printing.

**Recommended Print Specifications:**
- Paper size: 8.5" x 11" (Letter) for portrait flyers
- Resolution: 300 DPI minimum
- Color mode: RGB for digital, CMYK for professional printing (convert colors accordingly)
- Paper type: Glossy or matte cardstock for best results

### Digital Distribution

For digital distribution, the flyers can be used as-is or resized as needed.

**Platform Recommendations:**
- Email attachments: Use original size
- Social media: Use square flyer or resize portrait flyers to platform specifications
- Website: Optimize file size while maintaining quality
- Digital displays: Scale to appropriate resolution

### Brand Consistency

All flyers maintain consistent ICDFA branding elements with official colors.

**Color Usage:**
- Navy Blue #0D3265: Primary backgrounds, main text, headers
- Bright Cyan #07ADF6: Accents, highlights, call-to-action elements, borders
- White #FCFDFD: Primary text on dark backgrounds, clean sections
- Dark Brown #1A130A: Subtle accents and depth (minimal use)

**Branding Elements:**
- ICDFA identity and official colors
- Program Director: Aminu Idris, AMCPN
- Certifications: CCNA, CompTIA Security+, CEH, OSCP, CISSP, MPCSEAN
- GitHub repository: github.com/icdfa/ccc-program
- Application opening: January 4th, 2026

### Customization

If you need to customize these flyers, please maintain brand consistency.

**Guidelines:**
- Use only official ICDFA brand colors
- Maintain the core message and branding
- Update contact information as needed
- Ensure GitHub URL remains accurate
- Keep application opening date prominent
- Preserve ICDFA logo and identity

# Module 1 Quiz: Strategic Security Consulting

## Instructions

This quiz covers the key concepts from Module 1. It consists of 10 multiple-choice questions. Please answer to the best of your ability. The answer key is provided at the end.

---

### Questions

**1. Which of the following is NOT a core responsibility of a cybersecurity consultant?**
   a) Providing expert advice and guidance
   b) Implementing all recommended security controls
   c) Assessing security risks and vulnerabilities
   d) Developing security strategies and roadmaps

**2. What is the primary purpose of a risk appetite statement?**
   a) To eliminate all risks
   b) To define the amount of risk an organization is willing to accept
   c) To transfer all risks to a third party
   d) To document all known vulnerabilities

**3. In a qualitative risk assessment, how is risk typically calculated?**
   a) Risk = Asset Value × Exposure Factor
   b) Risk = Likelihood × Impact
   c) Risk = Threat × Vulnerability
   d) Risk = Annualized Loss Expectancy

**4. Which of the following is a key benefit of aligning security strategy with business objectives?**
   a) It guarantees that no security incidents will occur
   b) It ensures that security investments support business goals
   c) It eliminates the need for security audits
   d) It reduces the cost of security controls to zero

**5. What is the primary focus of the "Identify" function in the NIST Cybersecurity Framework?**
   a) To implement security controls
   b) To detect security incidents
   c) To develop an organizational understanding of cybersecurity risk
   d) To respond to security incidents

**6. Which of the following is a key component of a security roadmap?**
   a) A list of all employees and their roles
   b) A prioritized set of security initiatives over a multi-year period
   c) A detailed technical configuration guide for all security tools
   d) A log of all security incidents

**7. What is the primary difference between a policy and a standard?**
   a) Policies are mandatory, standards are optional
   b) Policies are high-level statements of intent, standards are specific mandatory requirements
   c) Standards are high-level statements of intent, policies are specific mandatory requirements
   d) There is no difference

**8. Which of the following is a key characteristic of a successful cybersecurity consultant?**
   a) Strong technical skills only
   b) Strong communication and business acumen
   c) Ability to work in isolation
   d) Focus on short-term technical fixes

**9. What is the primary purpose of a cost-benefit analysis for a security control?**
   a) To determine if the control is technically feasible
   b) To determine if the cost of the control is justified by the risk reduction it provides
   c) To select the most expensive security control
   d) To meet compliance requirements regardless of cost

**10. Which of the following is NOT a core function of the NIST Cybersecurity Framework?**
   a) Protect
   b) Detect
   c) Respond
   d) Mitigate

---

## Answer Key

1. b) Implementing all recommended security controls
2. b) To define the amount of risk an organization is willing to accept
3. b) Risk = Likelihood × Impact
4. b) It ensures that security investments support business goals
5. c) To develop an organizational understanding of cybersecurity risk
6. b) A prioritized set of security initiatives over a multi-year period
7. b) Policies are high-level statements of intent, standards are specific mandatory requirements
8. b) Strong communication and business acumen
9. b) To determine if the cost of the control is justified by the risk reduction it provides
10. d) Mitigate (The core functions are Identify, Protect, Detect, Respond, Recover)

# Module 1: Video Lectures

This document provides a list of planned video lectures for Module 1, along with links to relevant external videos that you can use as interim resources.

---

## Planned Video Lectures

- **1.1: The Role of the Cybersecurity Consultant**
  - Topics: Core responsibilities, ethical duties, value proposition, communication skills.
- **1.2: Business Risk Management**
  - Topics: Qualitative vs. quantitative risk assessment, risk appetite, risk treatment.
- **1.3: Security Strategy Development**
  - Topics: Aligning security with business objectives, creating a security roadmap, cost-benefit analysis.
- **1.4: Foundational Security Frameworks**
  - Topics: Overview of NIST CSF, ISO 27001, and other relevant frameworks.
